## Built With
- Nestjs
- Postgresql
- Docker
- Jest
- Swagger


## Prerequisites
- Node ^18.16.0
- Docker ^25.0.3


## Installation
1. Clone this repo
```
git clone https://github.com/pvdevs/crud-nestjs.git
```
2. ```cd``` to the project folder
3. now run
```
npm install
```

## Usage
Now run:
```
docker compose up -d
```

And then run:
```
npm run start
```

## Tests
Run:
```
npm run test
```

## Documentation
You can check out the documentation in:
```
localhost:3000/api
```

**If everything is correct, you should se a Hello World on your localhost:3000, and an empty array on localhost:3000/tasks**
